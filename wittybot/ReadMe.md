[WittyBot](https://grahamlehr.github.io/wittybot/WittyBot-Setup-FrameworkDependent-1.0.0.exe), is a Windows app that advances slides on voice commands using local Whisper. Built as an experiment. It needs .Net, but the installer should sort that out for you if you don't already have it.

Probably best not to rely on this in an actual show to advance slides. 🫣
